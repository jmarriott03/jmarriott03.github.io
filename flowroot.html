flowroot

display: flow-root is a CSS property value that creates a new Block Formatting Context (BFC) for an element without requiring hacks like overflow: hidden or clearfix. It forces the container to enclose its floated children, preventing them from "poking out" of the bottom, while also preventing internal margin collapsing with outside elements.

Use display: flow-root when:

  • A container has floated children and collapses
  • You want to avoid clearfix hacks
  • Margins are collapsing in confusing ways
  • You’re building reusable components (cards, media objects, content blocks)

Cleaner than alternative BFC triggers

Other ways to create a BFC include:

  • overflow: hidden
  • overflow: auto
  • display: inline-block
  • position: absolute

These often have side effects:

  • Clipped content
  • Scrollbars appearing
  • Inline behaviour you didn’t want
  • Positioning changes

display: flow-root gives you a BFC with no side effects

It exists specifically to solve these layout issues cleanly.

When you probably don’t need it

You usually don’t need it if:

  • You are already using Flexbox or Grid
  • You control all children and no floats are involved
  • You actually want margin collapse (rare, but it happens)
