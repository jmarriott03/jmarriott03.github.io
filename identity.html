Accessibility

We follow the Web Content Accessibility Guidelines (WCAG) 2.0.

https://www.w3.org/TR/WCAG20/

Choose font styles and sizes that are easy to read

The font size used in a document can be difficult for users with visual, print, and cognitive disabilities to read. Sans-serif fonts may be easier to read for some users. When these fonts are not used it may be difficult or impossible for the user to read the font.

https://www.webaccessibility.com/best_practices.php?best_practice_id=2148

http://webaim.org/techniques/fonts/#readability

For online reading, sans-serif fonts (e.g. Arial, Verdana) are generally considered more legible than serif fonts (Times New Roman), narrow fonts or decorative fonts.
