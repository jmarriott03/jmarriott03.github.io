Notes

h tags to have 'balance'.

h1, h2, h3, h4, h5, h6 {
text-wrap: balance;
margin: 0 0 24px 0;
}

p tag still 'pretty' and is full width of container.

p, ul, ol {
text-wrap: pretty;
max-width: 75ch;
margin: 0 0 24px 0;
}

Highlighted components can have full width background color.

Offer layout options for one-third, one-half, two-thirds, and full-width sections, all left-aligned within an 80em container.

w-1/3 container should expand to full width below 40em for improved responsiveness.

It is better to use max-width instead of width so that content can adapt more effectively to smaller viewports.

.w-1\/3 {
max-width: 26.666666em;
}
.w-1\/2 {
max-width: 40em;
}
.w-2\/3 {
max-width: 53.333333em;
}
@media (max-width: 40em) {
.w-1\/3 {
max-width: 100%;
}
}

Responsive padding 2em - 1em agreed.

Blue links only underlined on hover/focus.

.bg-color a {
color: var(--link-color);
text-decoration: none;
}
.bg-color a:hover, .bg-color a:focus {
text-decoration: underline;
}
/* Link behaviour override for mono theme only */
[data-theme="mono"] .bg-color-light a {
text-decoration: none;
}
[data-theme="mono"] .bg-color-light a:hover, [data-theme="mono"] .bg-color-light a:focus {
text-decoration: underline;
}

Dark theme has additional underline for 'None' background.

/* Link behaviour override for dark theme only */
[data-theme="dark"] .bg-color a {
text-decoration: underline;
}
